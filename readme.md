# WoW 亞服 安全認證指南 

## 準備條件
* 中國護照
* 科學上網環境(trojan|trojan-go|v2ray|vpn|ss 任意虛擬組網協議)
* 有網費有流量 有Money
* 未綁定任意戰網的郵箱地址和郵箱密碼
